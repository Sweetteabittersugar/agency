# 搜索优先

## 用途
在修改或新增代码前，强制执行搜索流程，发现现有实现、避免重复造轮子，保持代码库一致性。

## 核心规则
- 改任何文件前，先用 grep/glob 搜索相关函数、类、常量是否已存在
- 发现类似实现时，优先复用而非重写
- 搜索范围覆盖整个项目，不限于当前目录
- 搜索结果为空时再创建新代码
- 大范围修改前，搜索所有引用点，评估影响面
