# SPONSORING.md — 赞助

## 为什么赞助

agency-kit 是一个完全开源的项目，所有代码、Agent、Skill、Rule 都免费使用。你的赞助帮助我们：

- 持续的日常生产验证（保持 Agent 和 Rule 的质量）
- API 成本覆盖（测试多模型调度）
- 开发新 Agent 和 Skill
- 维护文档和社区支持

## 赞助方式

目前支持以下方式：

- **GitHub Sponsors**: [github.com/sponsors/Sweetteabittersugar](https://github.com/sponsors/Sweetteabittersugar)

## 赞助等级

| 等级 | 金额 | 权益 |
|------|------|------|
| ⭐ Supporter | $5/月 | README 致谢名单 |
| 🌟 Advocate | $20/月 | 以上 + 优先 Issue 处理 |
| 💎 Sponsor | $50/月 | 以上 + 定制 Agent/Skill 开发建议投票权 |

## 不是赞助的方式

如果你暂时不方便赞助，帮助项目的其他方式同样重要：

- ⭐ 给项目点 Star
- 🐛 提交 Bug Report
- 📝 改进文档
- 🔧 贡献 Agent/Skill/Rule
- 💬 分享给需要的朋友
